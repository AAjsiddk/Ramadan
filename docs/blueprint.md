# **App Name**: Ramadan Reflections

## Core Features:

- Daily Reflection Entry: Users can input daily reflections, thoughts, and prayers related to Ramadan.
- Link Integration: Provide quick access links to helpful resources: Kilonotes, نجاتك بيدك, Ramadan Telegram bot, and Ramadan WhatsApp channel, with copy buttons.
- Generative Prayer Tool: AI prayer tool based on personal reflections to aid in prayer composition and intention.  The LLM will use a tool to choose relevant prayers.
- Progress Visualization: Display a chart that visualizes users spiritual progress during Ramadan using inputted reflections and prayers.
- Goal Setting: Set and track personal goals for Ramadan. Goals could include daily prayers read, or spiritual acts performed.

## Style Guidelines:

- Primary color: Warm gold (#D4AF37) evoking spirituality and reflection.
- Background color: Soft beige (#F5F5DC) providing a calm backdrop.
- Accent color: Deep olive green (#556B2F) to signify growth and harmony.
- Headline font: 'Alegreya', a serif font for an elegant, intellectual, contemporary feel.
- Body font: 'PT Sans', a humanist sans-serif that combines a modern look and a little warmth or personality.
- Use minimalistic icons to represent reflection, prayer, and goals.
- Clean, spacious layout with a focus on readability.
